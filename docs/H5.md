## Demo工程

Demo [下载](http://fpvideo.shenshiads.com/demo_zip/H5-adsetdemo.zip)

## H5对接流程

1.  注册《一门打包》账号 [https://www.yimenapp.com/developer](https://www.yimenapp.com/developer)
2.  添加神蓍广告模块:   
      一门 --> 我的应用 --> 应用配置 --> 现在选购 - 广告平台/流量变现 --> 神蓍广告
3.  在一门上打包，正式id出来之前可以使用对接文档内的测试id
4.  申请媒体和广告位 [http://www.shenshiads.com](http://www.shenshiads.com)
